### C++ DSA Project - Simple Store-Product Management System

Description:
This is a simple console-based C++ project that simulates a basic store management system using a singly linked list. It allows users to add, display, search, and delete products.

Features:
- Add product (id, name, price, quantity)
- Display all products
- Search product by id
- Delete product by id
- Simple menu-driven system
